# @platform/config

Shared configuration, design tokens, and ESLint rules for the platform.

## Contents

- **design-tokens.ts**: Colors, spacing, radii, fonts, and font sizes
- **tsconfig.json**: Base TypeScript configuration
- **tsconfig.strict.json**: Extended strict TypeScript configuration for packages
- **.eslintrc.js**: ESLint configuration
- **prettier.config.js**: Code formatting rules

## Usage

```typescript
import { colors, spacing } from "@platform/config";
```
